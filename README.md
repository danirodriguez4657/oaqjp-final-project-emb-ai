# Emotion Detection Web App

This repository contains the final project for the IBM Skills Network course.  
The project implements a web-based emotion detector using Flask and IBM Watson NLP.

## Project Overview
The application allows users to enter text and receive an emotional analysis including:
- anger  
- disgust  
- fear  
- joy  
- sadness  
- dominant emotion  

The app is deployed locally using Flask and includes:
- Frontend (HTML + JS)
- Backend (Flask server)
- Error handling
- Static code analysis with PyLint

## How to Run
1. Install dependencies  
2. Run `python3 server.py`  
3. Open: http://localhost:5000
